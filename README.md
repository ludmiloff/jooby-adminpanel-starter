# Jooby Admin panel starter

A [Jooby](http://jooby.org) starter project featuring Pac4j, Pebble and JDBI3

## note
Some fixes and hacks have applied in order to run this project on Java 9. Not tested on Java 8 or 7.

## run
```
mvn jooby:run
```
